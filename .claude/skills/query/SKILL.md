# Query

Answer from the wiki, not from guesses.

1. Read wiki/index.md to find the relevant pages.
2. Read them.
3. Answer concisely, citing pages as [[wikilinks]].
4. If the answer is substantial and reusable, offer to save it as a new page.

If the wiki doesn't hold it, say so plainly. Don't invent facts.
