# Pricing model

How we price. The [[guy]] agent reads this before writing any proposal.

> Example Brain content for the starter. The numbers are illustrative. Replace
> them with yours.

## Models
- **Value-based** (default for new builds): price on the outcome, not the hours.
  A workflow that saves 10 hours a week is priced against that value, not the
  time it took to build.
- **Hourly** (ad-hoc and support): only when the scope is genuinely open-ended.

## Reference bands
| Package | Scope | Range |
|---------|-------|-------|
| Starter automation | one workflow, one integration | $2,000 – $4,000 |
| Agent build | one agent, Brain + 2 sources | $6,000 – $12,000 |
| Full system | multi-agent + dashboard | $20,000+ |

## Rules
- Quote a range in discovery, a fixed number in the proposal.
- Third-party API costs, travel, and rush timelines are billed on top.
- A number said on a call is a starting point, not a commitment. Always confirm
  it against these bands before it goes in a proposal.
